# Arrays - Ficha de trabalho 1

Neste repositório encontra 5 exercícios.

Resolva cada exercício, inserindo o código na linha com comentário ```// *** Inserir resolução do exercício aqui```

Exemplo:

```csharp
// Exercício 1: declare e inicialize um array do tipo int chamado numeros com os valores 1, 2, 3, 4 e 5

int[] numeros = { 1, 2, 3, 4, 5 };
```

Ao terminar cada exercício faça commit com a mensagem ```Resolução exercício x``` (em que ```x``` é o número do exercício que resolveu)
